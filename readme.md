# AutoHyper

Install [hyper](https://hyper.is/) terminal in all distros

## Requirements

* [jq](https://github.com/stedolan/jq)
* curl
* git

## Install
```bash
  git clone https://github.com/Dpbm/autohyper.git
  cd autohyper
  chmod +x install
  ./install
```

then, just go to your distro menu, and search for `Hyper` :)
